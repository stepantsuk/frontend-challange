## navigator.onLine
**Свойство navigator.onLine в HTML** — это свойство интерфейса Navigator, которое возвращает логическое значение, указывающее, находится ли браузер в онлайн- или офлайн-режиме.

**Использование:** свойство применяется для определения текущего состояния подключения к интернету на уровне браузера. Это позволяет **создавать приложения, которые адаптируются к изменениям в доступности сети**, например, отображать уведомления или переключаться в офлайн-режим.

**Важно**: не стоит доверять значению true как гарантии доступа в интернет — свойство может возвращать true при подключении к локальной сети без выхода в интернет.

## API Network Information
**navigator.connection**
Позволяет разработчикам получать информацию о сетевом соединении пользователя. Это включает данные о типе соединения (Wi-Fi, сотовое и т. д.), эффективной пропускной способности и том, включено ли ограничение на использование данных.
Интерфейс NetworkInformation содержит методы и свойства, связанные с типом сети браузера.

**Свойства**
Некоторые свойства Network Information API:
- type — указывает на тип соединения, который использует пользователь. Возможные значения: Wi-Fi, сотовое, none, ethernet, bluetooth, other.
- effectiveType — даёт оценку эффективной пропускной способности соединения. Может возвращать значения: slow-2g, 2g, 3g, 4g.
- downlink — предоставляет оценку эффективной пропускной способности в мегабитах в секунду (Мбит/с).
- rtt (Round Trip Time) — указывает на оценку времени обратного пути соединения в миллисекундах.
- saveData — логическое свойство, указывает, включён ли режим «Сохранить данные» в настройках браузера.
