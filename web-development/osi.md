# OSI

Это модель, а не стандарт как TCP/IP.

Уровни:
1. Application layer - прикладной уровень;
2. Presentation layer - уровень представления;
3. Session layer - сессионный уровень;
4. Transport layer - транспортный уровень;
5. Network layer - сетевой уровень;
6. Data link layer - канальный уровень;
7. Physical layer - физический уровень.
