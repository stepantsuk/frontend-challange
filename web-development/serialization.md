# Serialization & Deserialization

Сериализация и десериализация - это процессы преобразования данных из одной формы в другую с целью сохранения или передачи информации между различными данными системами или программами. Эти процессы часто используются в программировании во время работы с сетевыми протоколами, базами данных или сохранении состояния объектов.

Сериализация - это процесс преобразования сложных структур данных (объекты, массивы, словари и тд) в более простой формат, который может быть сохранен в файле или передан через сеть.
Используется для сохранения состояния объектов или передачи данных между программами. Результатом сериализации может текстовая строка или бинарное представление, которое легко восстановить обратно в исходные данные.

Десериализация - это обратный процесс, при котором данные из текстового или бинарного представления превращаются обратно сложные структуры.
Исполняется для использования и анализа этих данных в программе.

Пример. Объект с данными сериализуется на одной стороне в удобный для сетевой передачи формат, например JSON или бинарный поток. Затем происходит передача. Потом данные десериализуются на другой стороне для дальнейшего использования.

Языки программирования представляют инструменты для С/Д. 
В JS это JSON, Blob (Binary Large Object), Form data. При десериалзации необходимо оборачивать в try catch.
